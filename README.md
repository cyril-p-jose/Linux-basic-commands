# Linux Basic Commands 
 
 
---

## 1. pwd
Shows the current working directory.

```bash
pwd
```
## 2. ls
Lists files and directories.
```bash
ls
```
## 3. ls -l
Lists files with detailed information.
```bash
ls -l
```
## 4. ls -a
Shows hidden files.
```bash
ls -a
```
## 5. cd
Changes the current directory.
```bash
cd folder_name
```
## 6. cd ..
Moves one directory up.
```bash
cd ..
```
## 7. mkdir
Creates a new directory.
```bash
mkdir new_folder
```
## 8. rmdir
Deletes an empty directory.
```bash
rmdir folder_name
```
## 9. touch
Creates an empty file.
```bash
touch file.txt
```
## 10. cp
Copies a file.
```bash
cp source.txt destination.txt
```
## 11. cp -r
Copies a directory.
```bash
cp -r folder1 folder2
```
## 12. mv
Moves or renames a file.
```bash
mv old.txt new.txt
```
## 13. rm
Deletes a file.
```bash
rm file.txt
```
## 14. rm -r
Deletes a directory.
```bash
rm -r folder_name
```
